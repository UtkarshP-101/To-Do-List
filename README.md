# To-Do List Web App

A simple To-Do list web application built using Flask and SQLite.

## Features
- Add tasks
- View all tasks
- Delete tasks
- Persistent storage using SQLite

## Tech Stack
- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
